3. 新增訂單流程

步驟 1：點擊「+ 新增訂單」按鈕

在頁面右上角找到藍色的「+ 新增訂單」按鈕並點擊。

📍 按鈕位置： 頁面右上角，匯出報表按鈕旁邊

步驟 2：填寫訂單資料

▲ 新增訂單表單（Modal 彈窗）

📝 基本資訊

  • 訂單號 — 自動產生（如 PO-2026-006）
  • 客戶 * — 下拉選單選擇客戶
  • 交期 * — 選擇交貨日期

📋 產品項目

  • 產品型號 * — 輸入如 B36, W3030
  • 數量 * — 輸入數字
  • 單價 USD — 輸入單價
  • 小計 — 自動計算

步驟 3：新增多個產品（選用）

如果一張訂單有多種產品，點擊「+ 新增項目」按鈕增加行。

步驟 4：點擊「建立訂單」

確認資料無誤後，點擊右下角的藍色「建立訂單」按鈕。

✅ 成功： 訂單會出現在列表中，狀態為「草稿」

4. 編輯訂單

操作方式

  1. 在訂單列表中點擊任一訂單行
  2. Modal 會開啟並載入該訂單資料
  3. 修改需要的欄位
  4. 點擊「更新訂單」儲存
⚠️ 注意： 已出貨的訂單無法編輯
