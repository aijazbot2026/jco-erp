🎭 Modal 彈窗框架

Modal 是覆蓋在頁面上的彈出視窗，用於新增/編輯資料。以下是 ERP 系統中的所有 Modal：

Modal 名稱 所在模組 HTML ID 觸發按鈕 主要欄位
新增訂單 訂單中心 order-modal + 新增訂單 訂單號、客戶、交期、產品項目
新增出貨 訂單中心-出貨追蹤 shipment-modal + 新增出貨 訂單、貨櫃、船運、ETD/ETA
裝箱規格 訂單中心-裝箱管理 packing-modal + 新增規格 產品、每箱數量、尺寸、重量
新增材料 採購中心 material-modal + 新增材料 材料名稱、類別、單位、價格
新增供應商 採購中心 supplier-modal + 新增供應商 名稱、聯絡人、電話、地址
新增用戶 Dashboard user-modal 新增用戶 姓名、部門、角色、密碼

🔧 Modal 通用結構

<div id="xxx-modal" class="fixed inset-0 bg-black/50 hidden items-center justify-center z-50">
    <div class="bg-white rounded-2xl shadow-2xl w-full max-w-4xl mx-4">
        <!-- Header -->
        <div class="p-6 border-b">
            <h3>Modal 標題</h3>
            <button onclick="closeXxxModal()">X</button>
        </div>
        <!-- Body -->
        <div class="p-6">
            <!-- 表單欄位 -->
        </div>
        <!-- Footer -->
        <div class="p-6 border-t">
            <button onclick="closeXxxModal()">取消</button>
            <button onclick="saveXxx()">儲存</button>
        </div>
    </div>
</div>
